Cloudflare Worker
Een gratis Cloudflare Worker lost het CORS probleem op voor live koersen.

Stap 1: Ga naar workers.cloudflare.com (gratis account)
Stap 2: Maak een nieuwe Worker aan
Stap 3: Plak de code uit de Vault app
Stap 4: Kopieer je Worker URL hier in

Gratis: 100.000 requests/dag — meer dan genoeg.
export default {
 async fetch(request) {
  const url = new URL(request.url);
  const ticker = url.searchParams.get('ticker') || 'AAPL';
  const r = await fetch(`https://query1.finance.yahoo.com/v8/finance/chart/${ticker}?interval=1d&range=2d`,
   {headers:{'User-Agent':'Mozilla/5.0'}});
  const data = await r.json();
  return new Response(JSON.stringify(data), {headers:{'Content-Type':'application/json','Access-Control-Allow-Origin':'*'}});
 }
};
